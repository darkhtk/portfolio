The Challenge

산업용 로봇을 VR로 안전하게 조종하기

Meta Quest 3 VR 컨트롤러로 산업용 로봇(Doosan A0912)을 실시간 원격 조종하는 시스템이 필요했다. 단순한 데모가 아니라 RoboDK SIM과 실 로봇 양쪽에서 동작해야 하고, 엑스포 현장에서 실제 관람객 앞에서 시연해야 했다.

핵심 제약: (1) 안전 — 사람과 로봇이 같은 공간에 있을 때 절대 사고가 없어야 함, (2) 실시간 — VR 입력의 지연이 30ms 이하, (3) 정확성 — VR 좌표계와 로봇 좌표계 매핑 오차 최소화.

해결: Unity VR ↔ Flask ↔ RoboDK Python API ↔ 실 로봇 전체 파이프라인을 단독 설계·구현. 4-스레드 UDP 브릿지, Deadman 스위치, SAFE_BOX, 특이점 방지, EMA 필터, 캘리브레이션 시스템까지. Flask 서버와 UDP 브릿지 전체를 Claude Code AI Agent 활용으로 구현.

AI로 산업용 로봇을
실제로 조종하다

이 프로젝트의 핵심: Flask 서버, 4-스레드 UDP 브릿지, 안전 로직 전체를 Claude Code AI Agent 활용으로 단독 구현.

10년간의 로보틱스·시뮬레이션 도메인 지식으로 AI에게 기술 제약(Jacobian·특이점·관절 리밋)과 실시간 통신 아키텍처를 정확히 전달하여 산업용 수준의 코드를 구축.

결과: AI Agent 활용으로 실제 산업용 로봇을 조종하고 부산 벡스코 엑스포 전시까지 성공한 실증 사례.

자체 IK 솔버 — RoboDK 의존 없이 단독 구현

IK 알고리즘 자체부터 자체 구축

엑스포 프로젝트(MetaArm)는 RoboDK API에 IK 해석을 위임했지만, 별도 R&D(ik_meta)에서 IK 알고리즘 자체부터 전체 파이프라인까지 모두 AI Agent 활용으로 단독 구현.

자체 IK 솔버 (RuntimeIK)

Jacobian 기반 6DOF, 관절별 Weight Override, External Feedback (로봇 측정값 seed), 타겟 Rate Limit (선속도 m/s · 각속도 deg/s)

JointDef JointLimit SolveFlags Revolute · Prismatic
IkMotion — Teaching 시스템

관절 상태를 녹화 → 파일 저장 → 재생하는 모션 파이프라인. Recorder · Storage · Player 분리.

Recorder → Storage → Player
안전 로직 (단독 설계)

6축 개별 관절 리밋 클램프, 동적 속도 계산 (변위 비례), 특이점 감지, deg/rad 혼동 방지 휴리스틱.

Why It Matters

Jacobian 기반 IK는 일반적으로 로보틱스 전문 개발자가 수주~수개월 걸려 구현하는 영역. 이 프로젝트는 그것까지 AI 협업으로 단독 구현했다. "AI 도구를 사용하는 시니어"가 아니라 "AI에게 수학 알고리즘까지 전달할 수 있는 시니어"라는 차별화의 직접적 증거.
